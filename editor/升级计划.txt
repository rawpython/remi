## 🚀 分阶段升级建议

---

### 📦 第一阶段：基础稳定 & UTF-8 支持

完成#### ✅ 修改所有文件读写为 UTF-8    

确保避免 `"UnicodeDecodeError: invalid continuation byte"` 错误：

```python
with open(save_path_filename, "w", encoding="utf-8") as f:
    f.write(compiled_code)
```

同时修改 `load()` 方法中所有文件读取为：

```python
with open(filename, "r", encoding="utf-8") as f:
    source = f.read()
```

#### ✅ 修复项目无法加载错误（防止语法乱码）

---

### 🧱 第二阶段：重构控件元数据结构

#### ✅ 原本每个控件写死，可以改为统一注册结构：

```python
COMPONENTS = {
    "Button": {
        "class": gui.Button,
        "props": {
            "text": "按钮",
            "width": "100px"
        }
    },
    "Label": {
        "class": gui.Label,
        "props": {
            "text": "标签"
        }
    },
}
```

然后你拖放时只需 `COMPONENTS["Button"]["class"](**props)`

### 🧰 第四阶段：属性编辑器改造

for name in dir(widget):
